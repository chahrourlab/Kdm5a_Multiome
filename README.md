# Multiome analysis
Downstream analysis of snRNA-seq and snATAC-seq data using Seurat and Signac.

## Dataset 
This dataset is generated by sequencing the nuclei extracted from the cortices of 4 wildtype (WT) and 4 Kdm5a knockout (KO) mice. The raw data is available on GEO (accession number: GSEXXXXXX).

## Pipeline 
1. Create a Seurat object using CellBender output
2. Filter cells based on number of genes, number of UMI, and mitocondrial proportion (RNA filtering)
3. Normalize (SCT transform) and integrate data
4. Dimension reduction (PCA, UMAP, TSNE) and clustering
5. Find Gene markers. Split/merge clusters if needed. Annotate clusters
6. Downsampling
7. Differentially expressed genes
8. Filter cells based on number of peaks, number of UMI, mitocondrial proportion, TSS enrichment, and nucleosome signal (ATAC filtering)
9. Peak calling using MACS2
10. Transferring label infromation
11. Dfferentially accessible peaks
12. Motif analysis
13. Gene activity
